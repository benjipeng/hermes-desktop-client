# Security policy

Please report vulnerabilities through this repository's private GitHub security advisory form rather than a public issue.

Issues in the upstream Hermes agent, gateway, or official desktop source should be reported to [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent/security).

Include the client release, upstream Hermes tag or commit, macOS version, and whether the gateway used sign-in or token authentication.
