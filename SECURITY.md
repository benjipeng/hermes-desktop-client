# Security Policy

## Reporting a vulnerability

Use GitHub's private vulnerability reporting feature for this repository. Please include the affected version, reproduction steps, and expected impact.

Do not publish authentication cookies, gateway tokens, private dashboard URLs, or screenshots containing private conversations in a public issue.

Issues in Hermes Agent, the dashboard server, gateway authentication, or agent tools should be reported to the [upstream Hermes security process](https://github.com/NousResearch/hermes-agent/security).
